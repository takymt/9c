# 1行の文字列として入力する

- キーボード入力を1行まるごと制御できる `gets` / `puts` 関数がある
  - `scanf`と違って`%s`で空白が取れない、みたいなことがない
  - 実際はBOFが起こるのでdeperecated -> `fgets(文字配列, 配列要素数, ファイルポインタ)`
- `fgets`
  - あらゆるものはファイルで表現される(UNIX哲学)ので、`stdio`ファイルポインタがある
  - 数字を入力する場合は`atoi`や`atof`を使用する(数以外は無視される)
- `strtok(str, sep)`
  - 文字列のトークナイザー
  - 内部状態を持つため、初回読んだ文字列を2回目以降指定しなくても覚えている
  - 文字列に対して破壊的変更を実施する
  - `sep`には区切り文字の集合が入る(`,\n`なら`,`と`\n`が区切り文字)
