# 最小限の分割

- 把握可能性と並行編集のためにファイルを分割すべき
- 宣言のみ書かれたファイル: ヘッダファイル
  - `stdio.h`とか
  - ↔ ソースファイル(`stdio.c`)
- ヘッダ分割: `#include "hoge.h"`
- 宣言は複数回できるが定義は1回しかできない
  - `extern`宣言で変数をexportする(ただし使わなくて良い)
  - 関数宣言はデフォで`extern`相当
- ヘッダーファイルの重複を防ぐ `ifndef` / `endif`
  - 同じ宣言を何度も行う必要がなくなる(最終的に1つのファイルになるのでコンパイルは1回でOK)
