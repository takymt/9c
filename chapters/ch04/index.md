# 数値の表記と計算

- printfのフォーマット
  - 数字は文字列リテラル、数値は計算に使用可能
  - 数値を表示するためには`printf`の出力変換指定子(%演算子)
  - 位置に基づいて埋め込みできる
  - `%`は`%%`、小数点は`%.1f`のように表記する
- 整数と実数
  - 10進数/8進数(0始まり)/16進数(0x始まり)
  - 浮動小数点方式(IEEE754): 実数の表現方法、正負の値を広く計算できるが遅い
  - 少数の正確な表記には`%f`を使用する、有効数字は`%.2f`のように`.n`をつける

## IEEE754について

- 構成
  - 1桁目(1bit)     : 符号部(0なら正、1なら負)
  - 2-9桁目(8bit)   : 指数部E
  - 10-32桁目(23bit): 仮数部M
- 2^E * Mで表現できる
  - Mは1.xxxとして表現される
  - 0に近いほど正確に表せる事が多い
